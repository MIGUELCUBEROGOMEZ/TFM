Este depósito de GitHub contiene 3 archivos de R Y los datos comprimidos con 7zip.

R_AnalisisFuncional contiene los resultados para el análisis de componentes principales funcionales.
R_FMM contiene los resultados de los modelos cosinor y FMM.
R_Saliva_Cuestionarios contiene los resultados de los índices paramétricos, el sueño, la saliva y los cuestionarios.
